                      # Leidke tööd

                      DANGER

                      Состояние: Ещё не разрабатывается

                      Создайте крутое резюме или портфолио для быстрого поиска потенциальных работодателей. Удобство инструмента в конструкторе резюме в режиме реального времени с любого устройства, с дизайном и оформлением. Резюме можно расшарить как PDF, Word или API (что позволяет размещать резюме на сторонних сайтах)

                      Всё время актуальная информация, везде.
